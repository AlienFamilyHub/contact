# 相侵相碍异家人 | Alien Family Hub

这是异家人门户页面，立项于2025年2月2日，使用 [VitePress](https://vitepress.dev/zh/) 构建。

## 快速入门

### 安装依赖

```sh
pnpm i
```

### 本地测试

```sh
pnpm dev
```

### 修改成员

应当从群共享文档中修改，将在线表格中输出的 JSON 粘贴到 `docs/.vitepress/data/members.json` 中。

### 推送

提交 Pull Request。
